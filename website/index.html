What it does

Local MCP server

13+ tools any MCP client (Claude Desktop, Claude Code) can drive. Loopback-only HTTP bridge for the extension and desktop app.

CV parsing

PDF, DOCX, TXT — parsed locally. No upload. Heuristic skill detection.

Job matching

0–100 heuristic score with matched / missing skills. Explainable, auditable.

Tailoring & cover letters

Free: structural drafts. Pro: AI-rewritten prose from verified CV facts only.

Form autofill preview

Captures form fields, previews the mapping, flags sensitive fields. Never submits.

Application tracker

Local SQLite pipeline: draft → submitted → interview → offer.

Privacy by design

Local-first

CVs and applications live on your machine. The free core makes zero outbound calls.

Loopback only

The HTTP bridge binds to 127.0.0.1. Optional bearer token. No remote host permissions in the extension.

You approve everything

Submission is always manual. Sensitive form fields are flagged for your review.

Export & delete

Your data is a SQLite file in a folder you control. Delete it anytime.
